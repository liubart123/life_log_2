Terms:
    static text - hardcoded text, f.e. text in buttons, menus, field labels.
    dynamic text - text that can be changes by user actions - field values.

Usage of text styles:
    Headline - emphasize static text
        Medium - title at the top of screen, like 'Home page'
    Title - emphasize dynamic text  
        Large - Title of cards in scrollable list of cards
    Body - dynamic&static text used to display data 
        Medium - values and labels for fields
    Label - static text for action elements
        Medium - text for buttons, menu items, major error message
        Large - text for FAB
